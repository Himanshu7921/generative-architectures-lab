# generative-architectures-lab

From-scratch implementations of generative model architectures.

This repository explores the evolution of generative models, starting from
basic probabilistic approaches to modern deep learning methods.

## What you'll find
- Clean implementations (no heavy frameworks)
- Focus on core math and intuition
- Step-by-step progression of models

## Models covered
- Gaussian Mixture Models
- Variational Autoencoders (VAE)
- Generative Adversarial Networks (GAN)
- Normalizing Flows
- Diffusion Models

## Goal
To deeply understand how generative models work by building them from scratch.
